# PBO_24063
PROJEK PBO UNTUK MEMENUHI MATA KULIAH PBO
Class adalah cetak biru (blueprint) untuk membuat objek.
Di dalam class terdapat:
Atribut → data / variabel
Method → fungsi / perilaku
Contoh di dunia nyata:
Class: Mobil
Atribut: warna, merk, kecepatan
Method: jalan(), berhenti()
class → cetak biru object
object → hasil dari class
__init__ → constructor
self → representasi object
method → fungsi di dalam class
